IP位址及AS號碼數量統計

IP Addresses and AS Numbers

網際網路通訊協定位址(Internet Protocol Address)是用來辨識網際網路上主機位 置所在，以固定且唯一的一串數字位址分配予主機。台灣IP位址核發數量，在 IPv4位址發放數量統計上，台灣居全球第15名; IPv6位址發放數量為全球第18名。 Internet Protocol Address is a numerical label assigned to each device participating in a computer network that uses the Internet Protocol for communication.
